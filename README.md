## Install

    cd ~
    git clone git://github.com/maxim/dotfiles.git Dotfiles
    cd Dotfiles
    rake

## Install OS X customizations

    cd ~/Dotfiles
    rake osx

## Make changes

    cd ~/Dotfiles
    # make changes
    rake

## Update dependencies

    cd ~/Dotfiles
    rake update

## Remove everything generated

Removes symlinks too, be careful

    cd ~/Dotfiles
    rake cleanup
